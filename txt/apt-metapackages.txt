apt - metapackages
==================

This allows entire sets of software to be installed by selecting only the
appropriate metapackage.

Check the examples below.

X server :

--------------------------------------------------------------------------------
apt-get install xorg
--------------------------------------------------------------------------------

Gnome :

--------------------------------------------------------------------------------
apt-get install gnome-desktop-environment
--------------------------------------------------------------------------------

KDE :

--------------------------------------------------------------------------------
apt-get install kde
--------------------------------------------------------------------------------

Compilers and build tools :

--------------------------------------------------------------------------------
apt-get install build-essential
--------------------------------------------------------------------------------
