vim - visual block mode
=======================

Select a block on a bunch of lines

--------------------------------------------------------------------------------
CTRL-v
--------------------------------------------------------------------------------

Visual block + Insert

--------------------------------------------------------------------------------
CTRL-v (select the block) I (...) ESC
--------------------------------------------------------------------------------

Visual block + Delete then Insert

--------------------------------------------------------------------------------
CTRL-v (select the block) c (...) ESC
--------------------------------------------------------------------------------

Visual block indent

--------------------------------------------------------------------------------
CTRL-v (select the block) >
--------------------------------------------------------------------------------

Visual block comment out

--------------------------------------------------------------------------------
CTRL-v (select the block) I # ESC
--------------------------------------------------------------------------------

Additional tag : vi
