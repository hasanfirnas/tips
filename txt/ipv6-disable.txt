IPv6 - disable
==============

Create or edit `/etc/modprobe.d/ipv6.conf` and set :

--------------------------------------------------------------------------------
alias net-pf-10 off
# Add this for 5.4 and later :
options ipv6 disable=1
--------------------------------------------------------------------------------

In `/etc/sysctl.conf`

RHEL5, RHEL6 :

--------------------------------------------------------------------------------
net.ipv6.conf.all.autoconf = 0
--------------------------------------------------------------------------------

Online activation :

--------------------------------------------------------------------------------
sysctl -w net.ipv6.conf.all.autoconf=0
--------------------------------------------------------------------------------

Debian Squeeze / Ubuntu 10.04 :

--------------------------------------------------------------------------------
net.ipv6.conf.all.disable_ipv6 = 1
--------------------------------------------------------------------------------

Online activation :

--------------------------------------------------------------------------------
sysctl -w net.ipv6.conf.all.disable_ipv6=1
--------------------------------------------------------------------------------


Stop and disable the service `ipv6tables`

--------------------------------------------------------------------------------
/sbin/service ip6tables stop
/sbin/chkconfig ip6tables off
--------------------------------------------------------------------------------

On RedHat-like distributions, set in `/etc/sysconfig/network` :

--------------------------------------------------------------------------------
NETWORKING_IPV6=no
--------------------------------------------------------------------------------
