vnc - use
=========

Server - same remote and local desktop

-----------------------------------------------------------------------------
x11vnc -display :0 -forever # package = x11vnc
-----------------------------------------------------------------------------

Server - different remote and local desktops

-----------------------------------------------------------------------------
vncserver :1 # package = tightvncserver
-----------------------------------------------------------------------------

Server - kill vnc

-----------------------------------------------------------------------------
vncserver -kill :1
killall x11vnc
killall vncserver
-----------------------------------------------------------------------------

Client

-----------------------------------------------------------------------------
vncviewer 10.33.52.21:0
vncviewer 10.33.52.21:1
-----------------------------------------------------------------------------

Automatic start with Gnome Desktop login

-----------------------------------------------------------------------------
gnome-session-properties
-----------------------------------------------------------------------------

Add entry for cmd :

-----------------------------------------------------------------------------
nohup x11vnc -display :0 -forever
-----------------------------------------------------------------------------

Cf : `~/.config/autostart/`
