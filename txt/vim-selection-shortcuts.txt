vim - selection shortcuts
=========================

Select a block if the cursor is inside the parenthesis block

--------------------------------------------------------------------------------
vib
--------------------------------------------------------------------------------

Select a curly brace block (very common on C-style languages)

--------------------------------------------------------------------------------
viB
vi{
--------------------------------------------------------------------------------

Select text between quotes

--------------------------------------------------------------------------------
vi" for double quotes
vi' for single quotes
--------------------------------------------------------------------------------

Cut a paragraph

--------------------------------------------------------------------------------
{ d}
--------------------------------------------------------------------------------

Additional tags : vi, shortcut
